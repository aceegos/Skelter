# SKELTER — deploy-ready prototype

Files:
- index.html
- app.js
- manifest.webmanifest
- netlify.toml

## Fastest deployment
1. Unzip this folder.
2. Drag the whole `skelter-deploy` folder into Netlify Drop.
3. Open the HTTPS address Netlify gives you in Safari.

It is also a static site suitable for Cloudflare Pages, GitHub Pages, or any normal web host.

## Current rules implemented
54-card deck; seven cards each; human starts game 1 and starts alternate thereafter; opening table card 2–9; one card per turn; follow suit, rank, or make ten; draw exactly one when no legal play; Ace or 10 transfers the House; House owner may play same-suit court cards; Jokers are wild and choose suit; SKELTER declaration required after ending a turn with one card; first player to shed the final card wins immediately.

This build uses the on-screen SKELTER button. Voice recognition is intentionally omitted for iPhone reliability in this prototype.
